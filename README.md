# Proyecto: Rotisería Avilés

Página web simple creada con HTML, CSS y JavaScript.

Qué contiene

- `index.html` — página principal con menú, sección "Nosotros" y formulario de contacto.
- `CSS/styles.css` — estilos personalizados (usa Bootstrap también).
- `JS/script.js` — validación simple del formulario y anclas suaves.
- varias páginas de producto (por ejemplo `Pizza.html`, `Milanesa.html`) ya presentes en el repositorio.
